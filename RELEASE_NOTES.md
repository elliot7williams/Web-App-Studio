# Web App Studio Release Notes

## 0.1.0

Initial public preview for building, testing, packaging, and publishing installable web apps across phones, tablets, TVs, desktops, feature phones, kiosks, and embedded browser devices.

Highlights:

- SwiftUI macOS editor for HTML, CSS, JavaScript, and generated web manifests.
- Live WebKit preview with device frames, rotation, safe-area simulation, and screenshots.
- Built-in and custom device profiles with compatibility scoring.
- Local LAN preview server with QR codes for real-device testing.
- ZIP, handoff bundle, USB sync, device test kit, screenshot pack, and publish preset exports.
- Readiness, performance, deployment, compatibility, App Store metadata, and publishing support.
