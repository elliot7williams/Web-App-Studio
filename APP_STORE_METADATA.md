## Description

Web App Studio is a macOS tool for designing installable web apps that can adapt across many screens and browser-based devices.

Build from a live HTML, CSS, and JavaScript workspace, preview your app inside practical device profiles, then export a static web app package with a manifest, optional service worker, install prompt helper, and project README.

Create experiences for modern PWAs, compact phones, tablets, desktop browsers, TV browsers, wearables, feature phones, kiosks, and Firefox OS-style web app environments. Device presets include viewport sizes, user-agent notes, input expectations, and layout guidance so you can design with the target context in mind.

Web App Studio is built for developers, prototypers, students, and makers who want a focused place to shape portable web apps before testing them on real hardware.

Key features:

- Live WebKit preview for your current app
- Editable HTML, CSS, and JavaScript templates
- Generated web app manifest
- Presets for mobile, tablet, desktop, TV, wearable, feature phone, and Firefox OS-style targets
- Custom viewport sizing
- Optional offline service worker
- Optional install prompt helper
- Exportable static web app folder
